The TAIR Clone Details

When you select a clone name in the Clone Search Results window, the Clone Details appear. Generally, line titles do not appear in this page if there are no associated data for the clone, as in this example:

Name

Name of clone, if one exists. If not, "unnamed" appear here.

Alias

Other names for this clone, if any. Each alias appears in double quotes to allow for alias names with spaces. At this time, all aliases are displayed. Eventually, a maximum of 10 will be displayed with an ellipsis (...), linked to a page with a complete alias list, after the last entry.

Keywords

A list of all keywords associated with this clone. Keywords are delimited with double quotes (") to allow for keyword phrases.

Insert Size

Insert size and units for the clone. Only one size is displayed.

Alternate Size

A list of alternate sizes, if any. No limit on the number displayed.

Vector Type

Type of vector used for this clone, if any.

Vector Name

Name of vector, if any.

Library

Information about the library in which the clone was made. This information is not displayed in the prototype release.

Map Locations

Locations of the clone in different maps. The viewer column of these data contain links to the map visualization tool.

Sequences

Sequence(s) of this clone. Eventually, a maximum of 10 will be displayed with an ellipsis (...), linked to a page with the complete sequence list, after the last entry.

Sequence Type

Type of sequence.

Sequence IC Accession

IC Accession number.

Sequence BLAST

A link to submit a BLAST query for the given sequence. Not available in this release.

AGI Clone Information

Information about the sequence status and annotation of the clone.

Clone Ends

Information about the ends of this clone.

Chimerisms

Information about the chimeric characters of this clone, if known.

Hybridization

Hybridization information of this clone, if it was used for physical mapping. The hybridizing clone or probe and the map to which the hybridization was mapped are listed. This may not be displayed in the prototype version.

Used as probe

Indicates whether this clone was used as probe in a hybridization experiment.

Attribution

Displays type, community name, reference, and date of the attribution for the clone.

Attribution name

A link to the community page.

Attribution reference

A link to the reference page.

Comment

Displays comment date, comment author, and the comment associated with this clone. A comment may have two parts--the short comment, displayed in italics, and the long comment, displayed in normal font. If both comment styles exist, a colon separates them (:). Eventually, a maximum of 10 comments will be displayed with an ellipsis (...), linked to a page with a complete comment list, after the last entry.

History

Displays date and history type (insertion and update info) associated with this clone, if any. Eventually, a maximum of 10 will be displayed with an ellipsis (...), linked to a page with the complete history, after the last entry.

History Date

A link to the full history page for this history record.

Images

A list of images associated with this clone, if any. Any number of image names will be displayed.
