Close-Up on Sequence Maps

The Arabidopsis sequence map, the AGI map, is always the top map when selected for display. The map is a graphical representation of a segment of the chromosome, identified by the numbers at the top of the map. To facilitate more refined positioning of entities displayed, a chromosome ruler is displayed below the chromosome location numbers.

The location of the segment in the chromosome is also illustrated by the red rectangle on the bar in the map menu:

The AGI sequence map currently contains three types of entities: Markers, Clones, and Coding regions. All displayed entities are hyperlinked to a detail page from the TAIR Database. Each type of entity is drawn above a bar of tick marks. Position of the tick marks indicate the start position of an entity above (indicated by grey arrows in the above diagram). To facilitate the display of multiple maps, a show/hide function was implemented, where only three rows of entities or all of the entities can be shown. Displayed entities and hidden entities are indicated by red or black tick marks, respectively.

See "Show/Hide Entities" for more information.

Markers that are not visible at the current zoom level are indicated by black tick marks. Visible markers have names attached to them.

Clones appear below the markers. The different olors indicate the AGI sequencing status. See "Sequence Map Color Key" for details.

When known, the orientation of the sequence is indicated by an arrow pointing in the corresponding direction.

Coding regions are indicted below the clones and are visible only at 200x and higher zoom levels.

Note: The name and position of the coding regions are from TIGR. Currently the coding regions are not hyperlinked to our database. This will be fixed soon. You may find temporary differences between the sequencing data displayed at TAIR and the displayed on AGI web sites because of varying update times.

You can browse or search the sequence map using the map controls to the left of and above the map. For help with this, see "Using the Individual Map Controls".

You can also browse or search all of the displayed maps using the central controls at the top of the window. For help with this, see "Using the Central Controls".

One kind of Sequence map is available at this time: the AGI map.
