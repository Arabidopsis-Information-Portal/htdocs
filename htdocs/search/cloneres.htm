Using the Clone Search Results

When you submit a clone search, the Clone Search Results window appears. At the top of this window are additional search options..

New Clone Search by Name

Lets you perform a name search that is unconnected to the current search results. Wildcards are accepted (see "Using Wildcards"). After you type in the clone name, select the "go" button or press <Return> to submit the search request.

Search Again

Lets you return to the Clone Search window, with all search parameters defined as for the current search. You can then modify the search parameters and resubmit it. In future releases, this option will be called "Refine search" and will let you further restrict the previous search.

Download

Lets you download the full search results data set in a tab-delimited text format. This option is not available in the prototype release.

The remainder of the Clone Search Results window contains the search results data.

Summary

The Summary indicates the total number of records found in the search and the range within that total in the current display. If the results span more than one page, options at the bottom of the window let you move from page to page.

Previous

Accesses the set of results that fall immediately before the current set; e.g., if results 51 - 75 are currently shown in the window, results 26 - 50 will appear after you select previous.

Next

Accesses the set of results immediately following the current set; e.g., if results 51 - 75 are currently shown, results 76 - 100 will appear after you select next.

Numbered links

Let you jump forward or backward to a specific portion of the results list; e.g., if the page range is 50 records and you select 3, records 101 - 150 will appear.

Search Results

A numbered list of search results comprise the middle portion of this section. Each numbered line is a record. Records are numbered relative to the entire query results set so, for example, the second page of the query results shown above would contain 26 - 50.

Clone Name

The name of the clone that matched the search criteria. Each name is linked to detailed information about the clone.

Aliases

Synonym of the clone name.

Vector Type

The type of vector used in the clone, if it has one.

Chrom

The chromosome on which the clone is located.

Position

The location of the clone on the chromosome.

Map View

A link to a display of a map containing the cloned in the map visualization tool.
