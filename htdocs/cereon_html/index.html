CEREON ARABIDOPSIS POLYMORPHISM COLLECTION

This page provides access to a collection of predicted Arabidopsis single-nucelotide polymorphisms (SNP) and small insertions/deletions (INDELs) generated by Cereon Genomics. The data has been kindly submitted by Dr. Steve Rounsley.

New Releases There are now 28,117 SNPs, 15,674 INDELs, and 704 Large INDELs.

This data is accessible only to non-profit institutions, universities, and colleges. In order to access the data you must first fill out a one-time Registration form. Registered-users must then Login to access the data.
