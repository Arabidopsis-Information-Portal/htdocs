JIC ESTs to CIC YACs


NEW DATA ADDED ON 21 JULY, 1997

Data on 99 new clones were added on 16 July, 1997. These have been appended to the bottom of the table given below. Click here if you want to view only the new data. All other links on this page are to the complete (original plus update) version of the table.

The table given below shows the results of hybridisation of individual ESTs to colony filters of the CIC YAC library. The EST CloneID and it's Gen BankID are shown followed by the YAC co-ordinates identified. The data have been generated by colony hybridisation alone and have NOT been confirmed by Southern analysis. These preliminary data are presented because of their potential use to the Arabidopsis community, but please be aware that up to 10% of the co-ordinates may represent false positives.

Links are provided from each EST clone to GenBank and to the Arabidopsis EST analysis database at the University of Minnesota. All the EST and YAC clones are available from the Arabidopsis Biological Resource Center (ABRC). The EST CloneID can be used to search AIMS to order ESTs from there.

The raw-data file in tab-delimited format, which should be importable into most spreadsheet programs on most platforms, can be obtained using anonymous ftp from genome-ftp.stanford.edu in the directory /arabidopsis/EST2YAC. The file can can also be downloaded in Mac Excel format. If you are going to do serious searching of the table, it is recommended that you download the file. (This has the additional advantage of allowing you to reformat and sort the data.)

ESTs were kindly provided by the Ohio stock centre and Tom Newman. Chromosomal locations for the YACs were assigned by reference to Schmidt et al., 1996 (Science 270, 480-483) for chromosome IV, the Goodman Lab WWW server at MGH/Harvard for chromosome II, The Arabidopsis thaliana genome centre at the University of Pennsylvania for chromosome I and other chromosomes. We gratefully acknowledge the availability of the data in the CIC database managed by David Bouchez (INRA Versailles).

Data preparation for web display was done by TAIR.
