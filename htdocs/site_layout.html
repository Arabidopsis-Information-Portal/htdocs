The Basic Layout of TAIR's Web Site


Information within TAIR is in two distinct sections: the Illustra(Informix)-based database (Database Table of Contents), or TAIR Database proper, and separate web pages. In general, information in the database is not found in the web pages and vice versa. The TAIR project is gradually migrating data from the web pages into the database.

The TAIR Database proper includes interactive physical and genetic maps, the latest AGI sequencing information, colleague details, literature, clone and locus data.

The TAIR web pages provide tools and important information relevant to the Arabidopsis community and to the general plant biologist. These include, but are not limited to, BLAST, FASTA, Pattern Matching, Restriction Mapper, CAPS marker tables, the Weeds World electronic journal on Arabidopsis, the Electronic Arabidopsis Information Service journal (EAIS), Arabidopsis BioSci newsgroup archive, conference listings, splice-site tables, and a collection of protocols called the Compleat Guide. These have all been indexed using a web-search engine and so may be searched as a whole. Please note, however, that the database cannot be indexed in this fashion.

If you're not sure where the information you're looking for may be, either start at TAIR's Search Page or click in one of the categories displayed on the left of the home page. This will display links to pertintent information: in AtDB's database, on TAIR's web pages; and from external resources.

To get back to the home page, simply click on the leaf logo, which is found at the top left-hand corner of each TAIR Web page.
