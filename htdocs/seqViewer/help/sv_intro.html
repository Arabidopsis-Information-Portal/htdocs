TAIR's SeqViewer is a tool for viewing the Arabidopsis genome and its associated annotation from a whole genome view down to the nucleotide level. The SeqViewer displays gene annotations, annotation units (BAC clones used to generate and assemble the genome sequence), transcripts (including ESTs and full length cDNAs), polymorphisms, T-DNA/Transposon insertions, and markers on the Arabidopsis genome sequence. Users can search with up to 250 names or up to four short (< 150 nucleotide) nucleotide sequences, and visualize the locations of one or many search hits on the whole genome, in a closeup view (zoomable from 50 megabases to 10 kilobases) or in a 10 kilobase nucleotide window. Objects displayed in the SeqViewer are clickable and linked to TAIR database where you can find detailed records.

Note : The SeqViewer displays an annotation unit band to more accurately reflect the data used to construct the map. See theSeqViewer Release Update Notes for more information about the annotation unit band. Information about the version of the genome currently being displayed is shown on the SeqViewer home page. Coordinates can change between versions due to changes in the tiling path or BAC sequences.If you come across any bugs or errors in the tools or data shown here, please contact us at: curator@arabidopsis.org

SeqViewer Search Page

The SeqViewer entry page displays a whole genome view of the five Arabidopsis chromosomes and provides the following functions:

  • Search by name to find the location of a gene, annotation unit, polymorphism, transcript, marker or T-DNA/Transposon.
  • Search by sequence to find the location of up to four short sequences (15-150 nt each).
  • Open a closeup view by clicking on a point on one of the five chromosomes.

SeqViewer Closeup Page

The Closeup View page displays a whole genome view of the five Arabidopsis chromosomes at the top and one or more closeup views below containing marker, polymorphism, T-DNA/Transposon, gene model,transcript and annotation unit bands. Functions available from the closeup page include:

  • Scroll left or right, change zoom level, enter a range or recenter to customize the view.
  • Search within a single chromosome for a marker, gene, clone, .
  • Click on names of objects such as genes, T-DNA/Transposons, markers) to link to a TAIR detail page for more information on the object such as stock information for clones and insertion lines.
  • Open additional closeup views by clicking on another point in the five chromosome view at the top.
  • Open a nucleotide window to view 10 kb of nucleotide sequence with genes and other features highlighted.
  • Add or Remove sets of data from the closeup view.
  • Zoom in and out from a whole chromosome to a 10Kb view.
  • Show all data or only display three rows at a time for each data type.Hidden data will display as tickmarks.
  • List genes in range displayed in the view.

For more information on how to use the SeqViewer Closeup Page, click here.

SeqViewer Nucleotide Page

The Nucleotide View page displays a 10,000 nucleotide section of sequence displaying a customizable variety of objects annotated on the sequence. Functions available from the nucleotide page include:

  • Scroll left or right to jump 5000 nucleotides to the left or right.
  • View complement strand and show or hide annotations on the other strand.
  • Click on names (e.g. gene, marker, transcript) to link to detailed information from TAIR database.
  • Select one or more object types to be highlighted on the sequence.
  • Copy and paste a region of sequence into another application.
