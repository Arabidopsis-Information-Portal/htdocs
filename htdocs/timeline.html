This page provides information about TAIR's proposed timeline for release of major developments. Any part of this plan is subject to change in response to new/changed constraints, advice, and/or developments in related areas.

November 1, 1999 (projected) November 3, 1999 (achieved)
All of AtDB's static Web pages (flat file data and external links), graphic displays (physical map, sequence map, genetic maps), and sequence analysis programs (Pattern Match, Restriction Analysis, BLAST and FASTA) will be hosted from TAIR.

January 15, 2000 (projected)
A preliminary version of TAIR containing AGI progress, Map, Gene, Community, and Reference data (~90% of AtDB's database data) and the functionalities of AGI registration, community information, and interfaces for these data will be released.

March 1, 2000 (projected)
New comprehensive map display of sequence, physical, and genetic maps will be released.

September 1, 2000 (projected)
First version of TAIR database (with additional data on Genes, Stocks, Alleles, and Phenotypes) will be released.
