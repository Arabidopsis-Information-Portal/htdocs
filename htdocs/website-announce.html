New Website Release Notes

November 30, 2000

We are pleased to announce TAIR's new website. In addition to a complete redesign of the layout for the website, we have implemented several new features.

Features of our redesigned site include:

  • Easier navigation between web pages.
    - Toolbar lets you move between pages with fewer intermediate links.
  • Links between related pages are more logical, making it easier to find information.
    - Major site categories are AGI Information, External Links, Arabidopsis Information, News, and Tools.
  • Access to TAIR database development documentation and schema.
  • New Features - Searchable FAQs, Extended help pages, Job Postings, and Community Search.

We have also updated our Database and MapViewer. Please see the release notes to get the latest information on these tools.

  • Database Release Note.
  • MapViewer Release Note.

Please note that we are working on improving functionalities for our new website. Please let us know if you find any glitches, suggestions for improvement, and/or any errors. You can contact us at: curator@arabidopsis.org.

Acknowledgement of AtDB's Illustra database:

    AtDB's Illustra Database went offline on November 17, 2000. We would like to thank Mike Cherry, Gail Binkley, and Mark Schroeder at SGD for keeping up AtDB's Illustra Database since the AtDB project ended.
