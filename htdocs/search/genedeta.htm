The TAIR Gene Details

When you select a gene name on the Gene Search Results window, the Gene Details appear. Generally, line titles do not appear in this page if there are no associated data for the gene, as in this example:

Name

This is the mnemonic naming used for gene names by researchers. Examples include AG (Agamous), and QRT1 (Quartet1). A symbol for a gene is designated when a gene has been published or if the name and symbol have been registered at Oklahoma State, currently maintained by David Meinke's group or in GenBank.
(http://mutant.lse.okstate.edu/genepage/genepage.html).

Aliases

Synonyms of the gene name.

ORF Name

An open reading frame (ORF) name from the Arabidopsis Genome Initiative (AGI) groups' annotations. Usually, the convention for naming an ORF in Arabidopsis has been using the clone name followed by a number suffix (e.g., F23H14.13). For chromosome arms that have been completely sequenced, a standard ORF name designation is used:
  • AT (Arabidopsis thaliana)
  • 2 (chromosome number)
  • G (for Gene)
  • 01130 (Number)
    Examples: At2g01130, AT4g00010

Product Name

Name of a gene product. Examples: ASPARTATE AMINOTRANSFERASE, CYTOPLASMIC ISOZYME 1, CHALCONE SYNTHASE.

For the genes that do not have a full name or symbol--largely the predicted genes (ORFs) from AGI sequencing and annotation)--the following product names have been used:

  • Hypothetical protein
    Gene models without any database matches.
  • Unknown protein
    Gene models with only EST matches.

Product Description

Description of the gene product(s). Most of this data in TAIR currently is from AtDB curation of literature information

Gene Structure Predicted

Indicates whether the gene structure has been predicted.

Has Full-Length cDNA

Indicates whether a full-length cDNA sequence for this gene is available.

Mendel#

The gene identifier in the Mendel Plant Gene Nomenclature Database (http://genome-www.Stanford.edu/Mendel/)

Verification Method

In this version of the TAIR database, this method is for primary gene structure (translation start, intron/exon boundaries).

Derived Markers

Any genetic markers that were derived from this gene.

In Clones

The clones that contain this gene.

Map Locations

Details about all mapped location for the gene, including links to the TAIR Map Viewer and further details.

Sequence Locations

Details about sequence locations for the gene, including a link to attribution data.
