ESTs to YACs


TABLE IMPROVED AND NEW DATA ADDED ON 25 APRIL, 1996

The table given below presents results of PCR mapping experiments of Arabidopsis ESTs onto YACs. In all cases the EST CloneID, it's GenBankID, and the YAC clone it anchors are shown. Also given, where available, are the YAC size, all the markers to which the YAC hybridizes (in the case of it only hybridizing to an end-probe, the contig is also given), and the highest nucleotide homology for the EST. Note that if the YAC is found in two different places, the locations are separated by a forward slash. For example yUP15D11: 2-6191 / 4-LD, BIO206, GA1.

Links are provided from each EST clone to GenBank and to the Arabidopsis EST analysis database at the University of Minnesota. All the EST and YAC clones are available from the Arabidopsis Biological Resource Center (ABRC). The EST CloneID can be used to search AIMS to order ESTs from there. Please note that, due to quality control reasons, a few of the ESTs are no longer available in GenBank or dbEST. These clones, which are marked with an asterisk (*) in the table, are, however, still available from ABRC. Please e-mail comments@lenti.med.umn.edu if you have questions about such clones.

The raw-data file in tab-delimited format, which should be importable into most spreadsheet programs on most platforms, an be obtained using anonymous ftp from genome-ftp.stanford.edu in the directory /arabidopsis/EST2YAC. The file can can also be downloaded in Mac Excel format. If you are going to do serious searching of the table, it is recommended that you download one of these files. (This has the additional advantage of allowing you to reformat and sort the data.) In this regard, please note that "Find" only works properly in tables with Netscape version 3 and above. "Find" will work, however, with earlier Netscape versions on the plain (no links, space-delimited) text-file.

Mapping data was obtained for chromosome IV from AtDB, for chromosome II from the Goodman Lab WWW server at MGH/Harvard, all other mapping data and the yUP clone sizes were obtained from The Arabidopsis thaliana Genome Center at the University of Pennsylvania. CIC clone sizes were obtained from David Bouchez and the EG and EW clone sizes were obtained from Hwang et al. (1991. Plant J. 1, 367-374) and Putterill et al. (1993 Mol. Gen. Genet. 239, 145-157).
