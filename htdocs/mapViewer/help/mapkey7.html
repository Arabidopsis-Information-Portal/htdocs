Physical Maps

The physical maps show a segment of the chromosome represented by a bar. The location of the segment is indicated by a red box frame on the locationi bar to the left of the map. Numbers beside the scroll arrows at the top of the map indicate the base pairs illustrated.

Markers are indicated by a vertical line on the chromosome bar. A blue line is a visible marker and has a name attached. Agray line indicates the presence of a marker that is not visible at the current zoom level.

Below the chromosome are contigs from different groups. Each contig shows the positions of probes in vertical lines. Probes that are genetic markers are colored

red. Horizontal lines below the probes are clones. Each clone type is colored differently

AGI sequences

  • Light gray: intention to sequence
  • Red: sequencing in preparation or in progress
  • Orange: preliminary or complete sequence but not annotated
  • Dark green: annotated in GenBank
  • Gray: status unknown
  • Blue: marker in the sequence

Non-AGI sequences

  • Bright green: BAC clone
  • Pink: P1 clone
  • Dark blue: TAC clone
  • Blue: YAC clone
  • Dark gray: Unknown clone type

The hybridization points are drawn as dark red dots on the clones. Framework contigs are placed on the chromosome relative to the RI markers they contain.

Contigs from other groups are placed on the map relative to the framework contigs using common probes as anchoring points. Common probes in different groups are indicated by red dotted lines. All clones and probes are hyperlinked to a page that describes features about that clone or probe. The group names on the left are hyperlinked to the group's own display of the map data.

(i) Tiling Path -- This is defined as a hybridization-based physical map where the order of the probes and the

hybridizing clones is known, but little or no size information is known about either the clones or the probes. The

distance between each probe is unknown and so the clones are displayed equidistantly. Therefore, the size of the

clones displayed only represents the extent of their hybridizations to the probes, and are NOT real sizes.

(ii) Fingerprinted Contig -- This map is constructed using fragment sizes from restriction digests of clones to

calculate likely overlaps. This map rarely has probe data and has to be located onto the chromosome through

common clones that have already been placed by other means, such as through additional hybridization data. Most

of the fingerprinted contigs use BAC clones from the Genome Sequencing Center (GSC) at Washington University

(Marco Marra and colleagues, unpublished data).

(iii) Clone-sized Tiling Path -- This is the same as the tiling path except the size of the individual clones in the

contig is known and the overall size of the contig can be estimated by dividing the sum of all the clone sizes by the

redundancy of the clones hybridizing to each probe.

(iv) Physical Map -- The sizes of the clones, most of the probes, and the distance between probes are known or

can be inferred. This category includes most of the smaller-scale chromosomal walks performed for positional

cloning of genes.

(v) Sequence Contigs -- This is the definitive and ultimate physical map, where the sequence is known for a

chromosomal region.
