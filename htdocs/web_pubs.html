This page provides links to Web pages that give the text of, or further information from, Arabidopsis publications. Please send URLs for additions to this page, especially where the web pages provide more information than the published article, to: curator@arabidopsis.org Please note that submitted links must not be to a journal's site and must include more than the abstract.

Information on many Arabidopsis publications can be found in TAIR database, in the Papers section. Owing to time-constraints on TAIR staff, this currently does not have many papers from the last year or so.
For recent publications, try the excellent PubMed from NCBI, which now has many more plant-related journals indexed than their earlier Medline. Older plant-related publications, however, may well not be in PubMed and so it's worth searching both PubMed and TAIR for that elusive reference.
