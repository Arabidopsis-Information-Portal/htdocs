Submit COMPLEX Contig MAP Details to TAIR


THIS FORM IS FOR YOU TO PROVIDE COMPLEX PHYSICAL-MAP CONTIG INFORMATION TO ATDB.

In this form, you tell us who you are, the name of your contig, and the relative positions of all the probes and clones in your contig. Complete this form if your data is probably best represented as the type of physical map shown below. If you have less detailed mapping-data (no physical relationships between markers, etc.), please fill in a Submit Simple Mapping Data form instead.

An introduction to TAIR's forms for submitting physical-mapping data is available, as is a longer explanation and help.

If you already have some hybridization data in some sort of computer format, which you think may obviate the need to complete some of this or the other forms, please read this first.

SECTION 3. YOUR DATA

In order to complete this section, you first need to work out the relative map positions (in kb) of all the probes and markers used in the contig. To accomplish this, you first need to locate the most northern (leftmost) end of the first CLONE in the contig and set this as 0 (zero) kb. Measure all distances from the right of this point as plus (this should be all points).
