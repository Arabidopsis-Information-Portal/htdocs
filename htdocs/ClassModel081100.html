AGIClone

Description:


    An AGI clone is a clone that is intended, sequenced, or being sequenced by the
    Arabidopsis Genome Initiatiative and has a sequencing status and registration history associated.
    2/14/00 sue This table contains the input form fields.

    Class Attributes:


    NAME TYPE INITIAL VALUE VISIBILITY
    seq_status varchar(20) Internal
    sequence status for the registration: 10=intention to sequence or unknown;
    20 =clone library in preparation; 30 =sequencing started; 40 =preliminary GenBank
    release; 50 =sequencing finished; 60 =annotated GenBank release; 90 =clone
    deregistered
    registration_date smalldatetime Internal
    agi_clone_id id Internal
    chromosome int Internal
    clone_name varchar(50) Internal
    contig_number int Internal
    is arbiturarily designated to group the AGI clones in each contig. This number is generated
    whenever there is an update of new clones and the number itself does
    not signify order.
    genbank_accession varchar(20) Internal
    group_name varchar(50) Internal
    AGI registration groups: ABI, ATGC, CSHL, Genoscope, JIC, Kazusa, PGEC,
    Stanford, TIGR, Wash_U, Heidelberg
    insert_size int Internal
    insert size in kb
    left_overlap_clone varchar(50) Internal
    The clone on the left (or north) of this clone relative to the chromosome. Overlap is
    by sequence match (BLASTN)
    left_overlap_size int Internal
    markers_in_clone text Internal
    Genetic markers that are assigned to this clone, either by sequence match or
    hybridization.
    note text Internal
    Any note or comment made by the register
    orientation char(1) Internal
    Orientation of the clone relative to the chromosome. Foward = left to right
    Reverse = right to left.
    right_overlap_clone varchar(50) Internal
    see left overlap clone
    right_overlap_size int Internal
    see left overlap size
    sequence_size int Internal
    size of sequence for the clone's insert in GenBank. must have seq status of
    50 or 60

    Inheritance:

      Clone
      |
      |____ AGIClone

    Associations:

      RegisterAGIClone is associated with AGIClone

Assignment

Description:


    This table contains information about the position of mappable elements in maps.
    02/08/00 sue

    Class Attributes:


    NAME TYPE INITIAL VALUE VISIBILITY
    assignment_id id Internal
    this uniquely identifies an assignment
    start_position float(4) Internal
    this is the start position of an assignment. it cannot be null. unit information

    comes from the map the assignment lies in. 02/08/00 sue
    end_position float(4) Internal
    this describes the end position of an assignment. if an assignment is a point, the

    end position is the same as the start position. if an assignment has a range, the

    end position can be null. 02/08/00 sue
    range float(4) Internal
    this attribute describes the uncertainty of an assignment. a value in this attribute allows
    the assignment to lie within it.
    is_derived char(1) Internal
    This says whether a metric assignement was derived from a relative position. Depending
    on a map type,
    either relative position or metric position is derived. Primary assignment comes
    from AssignSource table.
    Map types whose primary assign is relative in TAIR:AGI, fingerprint. Map types whose primary
    assign is metric in TAIR: physical maps (minus fingerprint), reference sequence,
    RI,
    classical, miRFLP, 2/9/00 sue
    left_assignment assignment_id Internal
    this describes the left assigned object of the same assignment type on the same
    map . left is reltative to the map. 02/08/00 sue
    right_assignment assignment_id Internal
    This contains the object of the same type and to the right of the assigned
    object in the same map. right is relative to the map . 2/8/00 sue
    left_distance float(4) Internal
    this is for the distance between the assignment and its left_assignment.
    the unit is the same unit as the map where the assignment lies. 2/8/00 sue
    right_distance float(4) Internal
    this is for the distance between the assignment and its right_assignment. the unit for
    distance is the same as the map the assignment is on. 2/8/00 sue
    orientation char(1) Internal
    orientation of the assigned object relative to the map it is on. NOT relative
    to the chromosome. 2/8/00 sue
    viz_rank int Internal
    this is to rank the assignments on a map for the map visualization tool.

    Class Operations:


    NAME RETURN TYPE OPERATION VISIBILITY
    getMappableElementSize External
    calculate_left_coordinate External
    calculate_right_coordinate External
    getLeftAssignment External
    getRightAssignment External
    calculateDistanceLeft External
    calculateDistanceRight External

    Expanded Class Operation Defintions

    Inheritance:

      TAIRObject
      |
      |____ Assignment
