Introduction

TAIR's database search web interface provides options for reviewing the data in the TAIR database. The current release allows searching and downloading genes, genetic markers, clones, publications,GO annotations, locus histories and Arabidopsis community data. In addition, researchers can add and update their own personal and organization profiles. For more information about the latest release, please see the Release Notes

If you find any problems finding information, bugs, or errors in the data, please contact us at: curator@arabidopsis.org Each web interface page has a set of header and footer features in common.

The TAIR logo at the top left is linked to the TAIR home page. The links at the top let you access other tools and information on the TAIR site. The quick search allows you to perform simple searching of either the database or website. For more information on specific tools and information on the TAIR web site, please refer to our website help.

At the bottom of each screen, the NCGR, Carnegie, and ABRC logos are linked to each organization's respective home page. A link at the center bottom lets you send email to the TAIR and ABRC curators. For DNA or seed stock information, please send a message to dnastock@arabidopsis.org and seedstock@arabidopsis.org, respectively. Questions about all other data types should be sent to: curator@arabidopsis.org
