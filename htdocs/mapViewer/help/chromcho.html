Using the Chromosome Chooser

The Chromosome Chooser is the entry-point of the TAIR Map Viewer, also known as MapViewer Home.

Using the options here, you can access any map in the database by selecting a section or searching for a specific location.

With the exception of the chromosome drop-down menu and the search option, the content of this window is different for each chromosome. The list of available maps and the colored selection bars are specific to the chromosome. Default maps are selected (indicated by the checked box to the right of the map name). The color of the selection bar indicates the type of map:

  • : sequence map
  • : genetic map
  • : physical map

Several landmark markers appear on each selection bar. A dot on a bar indicates a centromere.

See "Accessing Maps" and "Finding a Specific Location on Any Chromosome".

Accessing Maps

  • Select a chromosome from the drop-down list

A list of the available maps for the selected chromosome appears.

Some maps are already selected in the list. A checkmark appears in the square to the right of selected map names. You can deselect any of them and select any others you wish to see.

  • Select the squares to the right of the map names to select/deselect them
  • Move your mouse cursor to the chromosome map bars

As you move the cursor on the bars, yellow segments appear at the top and bottom of the area. When the chromosome area you wish to view is between the yellow segments.

  • Select the area by clicking your mouse

The selected maps appear, showing the segment you selected.

See also "Finding a Specific Location on Any Chromosome".

Finding a Specific Location on Any Chromosome

  • Type the name of any mappable element (e.g., genetic marker, clone, gene, ORF name, etc.) in the search box. Wildcard searching is available if you are not sure of the exact name of the entity or if you want to search more than one entity at a time. Four wildcard characters/combinations are supported. Examples of wildcard searching: *SNP*, SNP*, *SNP, DET?
*

Matches 0 or more characters

?

Matches exactly one character

\*

Matches the character "*"

\?

Matches the character "?"

  • Select Search

All maps on all chromosomes plus alias tables are searched for the entity. All maps containing a match will be shown, and the searched for object will be highlighted in yellow. If the entity searched for is contained on more than one chromosome, only the maps for one chromosome are shown (chosen by the order of the chomosome number, e.g. if chr 1 and 3 maps contain matches, only chr 1 maps are shown). A list of other chromosomes containing search hits will appear at the top of the page under the heading "Additional matches". Clicking on one of these chromosomes will take you to the hits on that chromosome. A middle page showing all the matches on more than one chromosome will be implemented in the next version. The names of the displayed maps appear in red in the map list at the top of the Map window.

If no match is found, the message "The entity you searched for is not found" appears in a new window. When this happens, check the spelling of the entity you entered and try again, or try searching with a wildcard.
