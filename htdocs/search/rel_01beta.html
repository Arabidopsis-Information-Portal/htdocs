TAIR Database Update Notes

October 1, 2001 

Go to database

We are pleased to announce an updated version of TAIR database and search interfaces. This version will allow you to search the database for genes, genetic markers, clones, maps, publications, and community. Each data type has advanced search pages, where you will be able to search using more parameters including map positions.

Please note that we are continuing to improve and refine the system. Please let us know if you find any glitches in the interface, suggestions for improvement, any errors in the data, or missing information by contacting us at: curator@arabidopsis.org.

Major Enhancements

  • Updated content
    • Gene data loaded from TIGR and AtDB.
    • Locus and gene model association.
  • Updated Search functions
    • Drop down menu for wild card searching.
    • Download search results.
    • Search Arabidopsis publications.
  • New functions
    • Order stocks from ABRC.
    • Update and Register Personal and Organization profiles.
    • Login.
  • Software
    • Consolidation of system architecture in Java.

Projects in Progress

  • Addition of gene and sequence data from the literature and GenBank.
  • Annotation of genes using controlled vocabulary such as Gene Ontology terms.
  • Search for Keywords, Sequence, Germplasm, Phenotype, Protein, Expression and Gene Families.
  • Browse data in tabular and expandable tree views.
  • Refined search option where nested queries can be performed.
  • Hyperlink detail pages to SeqViewer.
