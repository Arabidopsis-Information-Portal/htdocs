Using the Clone Search

The TAIR Clone Search window provides three ways of searching for a clone: simple search by name only, complex search using more limits, and search by position. These are offered in three distinct sections of the window.

Search by Name

At the top of the TAIR Clone Search window are the Search by Name options. The software searches all the names and aliases of Clones.

This simple search is case insensitive and allows wildcard searching (see "Using Wildcards"). Leaving the input box blank retrieves all Clone entries in the database.

Clone Name

Arabidopsis clones are usually named with the acronym of the vector followed by the plate and row numbers of the isolated clone. For example, CIC (YAC), T (TAMU BAC), F (IGF BAC) are some common vector acronyms.

Example clone names: CIC9A4, F16F4

You can select to display 25, 50, or 100 clone result items on a single page. More results per page will take longer to load.

Restrict by Features

The Restrict by Features options are below the Search by Name options on the TAIR Clone Search window. All the options in this section are an 'AND' operation. If the name input box is left blank, the software searches all the entries in the database matching the checked criteria.

These options let you perform more complex searches of several data types grouped into three parameters (Vector Type, Insert Type, and Other Features). You can also select to limit the search by time. You can select multiple types within each parameter.

You can select to search one or more vector types using the check boxes.

Other Features

Attributes that limit the search. Check boxes let you select one or more attributes. Selecting multiple attributes indicates an "AND" relationship. For example, checking "is sequenced" and "is ABRC stock" limits the query set to sequenced clones that are available as distributed stock from ABRC.

Insert Type

Types of insert DNA that clones contain. Check boxes let you select one or more types. Selecting multiple attributes indicates an "AND" relationship. T-DNA refers to T-DNA from Ti-plasmid used for transformation.

Time Restrictions

Lets you limit your search to the most recent entries only.

Restrict by Map Location

The bottom section of the window lets you restrict your search by location.

The options in this section let you use two parameters to restrict your search: Chromosome and Map Type.

Chromosome

Lets you limit your search to a single chromosome. There are five chromosomes in Arabidopsis: 1, 2, 3, 4, and 5.

Map Type

When implemented will let you search a particular map type. Until this is implemented, you can search multiple map types using the TAIR Map Viewer.
