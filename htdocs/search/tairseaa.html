Introduction

TAIR's database search web interface provides options for reviewing the data in the TAIR database. The current release is a prototype supporting only a limited number of tasks and information. We plan to release the first full version of the interface and database in the fall of 2000.

Each web interface page has a set of header and footer features in common.

The TAIR logo at the top left is linked to the TAIR home page. The screen title appear to the right of the logo. The graphic envelope at the top right lets you send mail to the TAIR curators. The question mark beside it accesses the help documentation for the page.

At the bottom of each screen, the NCGR and Carnegie logos are linked to each organization's respective home page. Links at the center bottom let you access other parts of the TAIR site.
