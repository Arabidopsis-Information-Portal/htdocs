The Carnegie Institution for Science, Department of Plant Biology, located on the Stanford University campus, is looking for a senior software developer to work on our computational programming interface (SOAP and REST web services, query tools, and database access). TAIR, The Arabidopsis Information Resource -- http://arabidopsis.org -- provides leading-edge genomics research data and tools. This position is part of the Plant Genome computational API grant from the NSF.

Responsibilities:

  • Programming software systems in Java and SQL
  • Deploying and managing the configuration of open-source code
  • Managing data in relational databases (Oracle and PostgreSQL) with SQL
  • Other duties as required

Requirements:

  • BS in Computer Science/Management Information Systems or equivalent experience.
  • At least two years of experience with Java programming
  • At least one year of experience with the database language SQL and a good grasp of relational database management
  • At least six months of experience programming Java-based SOAP and REST web services
  • Basic knowledge of software engineering (programming, testing, object-oriented systems programming and design)
  • Flexibility to learn and grow in a dynamic research environment
  • Ability to interact, to work cooperatively, and to communicate with co-workers

Preferred:

  • Knowledge of basic biology and genetics
  • Knowledge of web-based programming (HTML, Javascript)
  • Knowledge of the Java parsing system ANTLR
  • Experience with the git version control system

Carnegie offers excellent benefits, a competitive salary, 12 days of vacation and 11 holidays per year, a great work location on the Stanford campus, and a stimulating scientific and computing environment.
