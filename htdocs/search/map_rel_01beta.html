Map Viewer Release Update Notes

June 14, 2001

We are pleased to announce an update to TAIR's comprehensive Map Viewer. It allows you to search, browse, and align the map elements and maps available from TAIR's database. Each map element is hyperlinked to a detail page from our database. A detailed Help document on what is available and how to navigate this tool is available: TAIR Map Viewer Help

Major Enhancements :

  • Clicking on genes will display gene detail page from the database.
  • Displays hits on multiple chromosomes from text searching.
  • Searches through aliases.
  • Coding regions from TIGR/MIPS annotation are displayed on the AGI maps at zoom level 200x and higher.
  • Wild card searching is implemented and will highlight all entities found. (e.g. *SNP*, AG?, ve*)
  • Show/hide function is implemented to display three rows of entities or all entities.
  • Tick marks to show entities that are hidden.
  • Genetic maps are zoomable to 80x.

Bug Fixes :

  • All clones and probes on physical maps are shown.
  • Entities on the south ends of the chromosomes are shown.

Known Bugs :

  • Some probes on Physical maps are not currently hyperlinked to the database.
  • A few of the coding regions are not shown on the AGI map, due to discrepancy in the name of the clones or lack of annotation data from TIGR.

Coming Soon :

  • Show/Hide function according to data types for each entity.
  • Reference Sequence Map zoomable down to nucleotide level and displaying sequence annotations including gene features and polymorphisms.
  • Additional entities such as insertion sites and transcripts will be displayed.
  • Experimentally verified gene structures will be color-coded.
  • Different marker types will be color-coded.

Please note that this is a second version and we are working on improving functionalities for our next version. Please let us know if you find any glitches in the interface, suggestions for improvement, and/or any errors in the data. You can contact us at: curator@arabidopsis.org.
