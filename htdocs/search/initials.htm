Using the TAIR Search Page

The TAIR Search Page provides options you can use to search the TAIR database. You can do simple or complex searching. Simple searching is a name search of all data types or a specific data type. Complex searching uses additional parameters to search a specific data type. In the current version, complex searching is available for Clones, Genes, Genetic Markers, Maps, Publications, and Arabidopsis Organizations and Researchers. Simple search forms for specific searching of Protein data, Gene Ontology (GO) Annotations and Locus Histories is also available. The next version of the search software will include simple searching by keywords and complex searching of multiple data types.

General Search

At the top of the TAIR Search Page, are the General Search options.

General Search lets you search most TAIR data by name. This search is particularly useful if you have the name of an entity but do not know the data class to which it belongs. In the current version, this feature searches clones, genes, genetic markers, people's last names, organization names, and all aliases. TAIR's database contains many types of data, which are defined and described in detail in the TAIR Data Model and Documentation.

Using Wildcard Drop Down Menu

You can use the wildcard dropdown menu to the left of the search input box in your name searches. Four types of wildcard options are supported:

exactly

name matches input string exactly
contains

name contains the input string at the beginning, middle, or the end
starts with

name contains the input string at the beginning
ends with

name contains the input string at the end

Specific Search

The Specific Search options appear below the General Search options.

These options let you search within specific data types in TAIR. The data types (described below) are listed on the left. The search option appears to the right of each data type. When you select the search option for a data type, a search form appears. The specific search is most useful if you know the type of data you seek.

Data Types

Markers

This data type represents genetic markers that can be used to experimentally distinguish between polymorphisms. The types of markers are: CAPS, SSLP, AFLP, RFLP, RAPD, PCR based, and hybridization based. SNPs and visible markers (polymorphisms of unknown sequence variation but characterized phenotype) can now be found using the Polymorphism/Allele Search. Potential polymorphisms (e.g., dinucleotide repeats or satellites) are also included in this category for searching.

Genes

The genes data type represents any Arabidopsis gene model or locus. In TAIR, a Gene Model is defined as any description of a gene product from a variety of sources including computational prediction, mRNA sequencing, or genetic characterization. A locus is defined as the genomic sequence corresponding to a transcribed unit (e.g. AT2G03340) in the genome. In TAIR, many gene models can exist for a given locus, therefore a search for a gene may result in multiple hits for the same gene name

DNA

This data type represents any object that contains a piece of Arabidopsis DNA and can exist in the form of a clone, clone end, library, purified genomic DNA, filters. Many of these objects are sequenced and are available as DNA stocks from ABRC.

Polymorphism

Any genetic variation between two ecotypes or within one ecotype that can be assessed experimentally by sequencing, PCR amplification, restriction digest analysis or phenotypic inspection.

Clones

This data type represents any piece of DNA that is inserted into a vector.

Proteins

This data type represents molecules built from amino acids joined together with peptide bonds. Proteins are associated to physical, structural and other characteristics such as: isoelectric point, molecular weight,domains, motifs, and subcellular locations.

Sequences

This data type contains sequence text and information for all Arabidopsis nucleotide and amino acid sequences. Nucleotide sequence types include genomic, cDNA, oligos, polymorphic sequences, and sequences flanking molecular objects such as genetic markers (e.g., primers).

Maps

This data type represents any entity that has other entities assigned to (or positioned on) it. Maps are largely separated into genetic, physical, and sequence maps in TAIR. However, entites such as clones and genes will also be considered to be maps in TAIR, if the entities have other elements assigned to them.

Community

This data type includes the organizations and individual researchers who study Arabidopsis. Organizations may include Arabidopsis labs, stock centers, committees, facilities, universities, and non- and for-profit institutes.

Reference

This data type represents reference information for Arabidopsis data. The types of reference in TAIR include journal article, book, book chapter, thesis, personal communication, electronic references, conference proceedings, analysis reference (for software-analyzed data such as gene annotation), database reference (data from external databases such as GenBank and TIGR), and websites.

Publication

This data type represents publications for Arabidopsis data. Types of publication in TAIR include journal article, book, book chapter, thesis, conference proceedings, and press releases.

GO Annotations

Annotations of Arabidopsis gene products with controlled vocabulary terms for gene product molecular functions, cellular components and biological processes. Terms and ontology structures are developed by the Gene Ontology Consortium.

Locus History

A locus is defined in TAIR as a genomic sequence corresponding to a transcribed unit (e.g. AT2G03440) in the genome. Loci may change due to splitting or merging of two or more loci. Also, some loci may be deleted due to insufficient evidence and new ones inserted. The locus history data includes information about loci that have been merged, split,inserted or made obsolete. The locus history search can also be used to find a locus name that has not previously been used when naming a new locus.

Microarray

Array elements are the nucleotide molecules that are printed onto microarray slides or in situ synthesized onto a chip. In the case of the Affymetrix chips, the array elements are considered to be the set of oligonucleotides that define a target gene (probe sets).
