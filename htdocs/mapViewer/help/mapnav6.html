Search

You can search for a specific mappable element--a genetic marker, clone, gene, etc.--using the search options:

With the central controls you can select to search

  • All maps selected in the map list to the left
  • All maps available for the current chromosome
  • All chromosomes

To search for a mappable element

  • Select a search option from the drop-down list
  • Type in the name of the element you're looking for
  • Select Find

The window redisplays. The names of the matching maps appear in red in the map list at the top:

The matching element is highlighted in yellow in the maps. Any maps in the window that do not contain the element appear with a dark gray background.
