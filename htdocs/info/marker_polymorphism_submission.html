Marker and Polymorphism Data Submission to TAIR

Fields indicated in blue are required. Fields marked with an asterisk(*) can have more than one entry but the entries must be seperated by a semicolon with no intervening spaces (e.g. Ecker,Joseph;Bell,Callum). Please fill in as much information as possible.
Please include the column headers as the first line in your files.

Field Description Values/Constraints Example

polymorphism type Definitions of polymorphism types
digest_pattern:polymorphisms detected by restriction enzyme digestion
PCR_product_length: polymorphisms detected by PCR
SNPs: single nucleotide polymorphisms
INDELs: insertion/ deletions where sequence between two identical flank sequences is deleted in one ecotype relative to another
INSERTIONS: include native and introduced transposons, transgene insertions and T-DNAs
SUBSTITUTIONS: sequence variations where two ecotypes differ in sequence between identical flanking sequences only in base composition but not number.
DELETION: used for polymorphisms where one allele has one or more sequential base pairs deleted relative to an allele of the same ecotype.		 digest_pattern, PCR_product_length, SNP, INDEL, INSERTION, SUBSTITUTION, DELETION, UNKNOWN SNP
insertion type transposon, gene fusion, T-DNA, other limit to 50 characters transposon
DNA fragment length* length of digested fragment, PCR product length, polymorphism length or AFLP product length in bp, seperate multiple sizes with ; (no intervening spaces) limit 60 characters 1300
Flank Sequence type For CAPS, SSLP, AFLP and RAPD markers this corresponds to the PCR primers user to amplify the marker before digestion. For SNPs this corresponds to 20 base pairs flanking the site of the SNP on either side. For INSERTIONS/DELETIONS, this is the sequence flanking the site of insertion/deletion. PCR_primer flank
